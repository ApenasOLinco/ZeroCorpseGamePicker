# ZeroCorpseGamePicker
Um script que seleciona um jogo aleatório dentro da [página de jogos do site da ZeroCorpse](https://www.zerocorpse.com.br/p/download.html). Sim, extremamente específico.

## Uso
É só copiar e colar esse script no console do seu navegador (`F12` > console) enquanto você está na página do link acima. Será adicionado um novo elemento na página com o jogo selecionado. Boa jogatina :)
